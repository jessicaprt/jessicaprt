Tic Tac Toe

For my final project in CMPUT 275 - Introduction to Tangible Computing II, my partner and I decided to build a one-player Tic Tac Toe game.

This impossible game of tic tac toe was implemented by using the minimax algorithm (written in Python) in order to build an AI such that the computer will always win, or the game results in a draw.

Fire Escape

My first hackathon experience! Fire Escape is an arcade game that aims to educate players with fire hazards and safety. Four other competitors including myself created this project as part of an entry to hackEd 2016: U of A Computer Engineering Club's 24-hour hackathon. My part for this project is being in charge of the game's assets-- this includes most of the graphics used in the game, building the maps for the levels, and choosing the sound effects.

Snake on Arduino

This was my final project as part of my course in CMPUT 274 - Introduction to Tangible Computing I.

For the hardware components of this version of the classic snake game, a joystick and an LCD (Adafruit) was connected to an Arduino Mega. This game was written in C with the use of the Arduino library.

Personal Website

With my love for designing, I used what knowledge I have into building a website and played around with it to make it look the way it does. I expect this website to change its looks quite often as I discover more about web development.
