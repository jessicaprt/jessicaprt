Snake on Arduino

This was my final project as part of my course in CMPUT 274 - Introduction to Tangible Computing.

For the hardware components of this version of the classic snake game, a joystick and an LCD (Adafruit) was connected to an Arduino Mega. This game was written in C with the use of the Arduino library.
