Arduino Binary Clock

A binary clock is a clock that displays the time of day in binary format (wikipedia). Each column represents a single digit (i.e. first colums represents the tens place for hours and the second colums represents the ones place, etc..). 20 LED bulbs were used for display: 6 to represent the hour, and 7 each to represent the minute and seconds. This clock is set manually by using three pushbuttons: one to set up the hour, one for minutes, and one for resetting the seconds.

Tic Tac Toe

For my final project in CMPUT 275 - Introduction to Tangible Computing II, my partner and I decided to build a one-player Tic Tac Toe game.

This impossible game of tic tac toe was implemented by using the minimax algorithm in order to build an AI such that the computer will always win, or the game results in a draw. The game was written in Python with a basic GUI built with tkinter.

Snake on Arduino

This was my final project as part of my course in CMPUT 274 - Introduction to Tangible Computing I. This version of the classic Snake game (built with the Arduino Mega) was written in C with the use of the Arduino library.

A 1.8" TFT w/ MicroSD Adafruit LCD was used for the display. To control the movement of the snake, a joystick was used. As for the levels, the more "apples" the snake eats, the faster it moves. The game ends when the snake hits the wall or its body.
