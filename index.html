PROJECTS

Here are some of my projects that I've done in school as well as my other personal projects. Click here to view more.

Tic Tac Toe

This impossible game of tic tac toe was implemented by using the minimax algorithm (written in Python) in order to build an AI such that the computer will always win, or the game results in a draw.

Snake on Arduino

For the hardware components of this version of the classic snake game, a joystick and an LCD (Adafruit) was connected to an Arduino Mega. This game was written in C with the use of the Arduino library.
