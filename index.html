Projects

Here I have some of my projects that I have worked with during my university classes, as well as some personal projects that I have done.

Tic Tac Toe

This impossible game of tic tac toe was implemented by using the minimax algorithm in order to build an AI such that the computer will always win, or the game results in a draw. [Python]

Snake - on Arduino

For the hardware components of this version of the classic snake game, a joystick and an LCD (Adafruit) was connected to an Arduino Mega. This game was written in C with the use of the Arduino library.
